Eine entfernte (Remote) Kompilierungsumgebung

BVRDE ist eine integrierte Entwicklungsumgebung welche es erlaubt C und C++ Projekte auf einem entfernten UNIX oder Linux Server zu editieren, kompilieren und auszufühgren.
Integriert mit dem GNU C Compiler und GDB Debugger, entlastet es Sie von immer wiederkehrenden make oder debug Befehlen.

 

Zuletzt geöffnete Projekte 


Wo BVRDE sie unterstützt

  • Hübscher Editor mit Syntaxhilighting
  • Erspart Ihnen Zeit um sich auf die kreative und anspruchstvolle Programmierung zu konzentrieren
  • Einfaches kompilieren von Projekten - auf allen Plattformen
  • Support für debugging Befehle und Inspector
  • Gestattet mehrere entfernte Projekte in einem Arbeitsbereich
  • Ist mit der spizenmäßigen WTL Technologie erstellt
